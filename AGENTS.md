# AstraNexus Agent Documentation (AGENTS.md)

## 概要
本プロジェクト（AstraNexus コーポレートサイト）の開発・保守を担当しているAIエージェントの構成と、これまでに実施した開発作業の履歴を管理・ドキュメント化するためのファイルです。

## エージェント体制

### 1. メインエージェント: Antigravity
Google DeepMindチームによって開発された、高度な自律的コーディングアシスタントです。ペアプログラミングを通じてユーザーの要望を解釈し、コードの設計、リファクタリング、SEO対策、ビルド・デプロイ検証までをワンストップで実行します。

* **得意領域**: モダンWeb開発（HTML5, CSS3, JavaScript/TypeScript）、SEO/パフォーマンス最適化、リファクタリング
* **通信・操作**: ファイル読み書き、ターミナルコマンド実行、画像生成、サブエージェント起用

### 2. サブエージェント
複雑なタスクを効率的に並行処理するため、必要に応じて以下のサブエージェントを召喚します。

| エージェント名 | 役割 | 主な役割・機能 |
| :--- | :--- | :--- |
| **research** | コードベース調査・情報収集 | 読み取り専用権限を持ち、広範なコード検索やWebからのドキュメント調査を担当します。 |
| **self** | メインプロセスのクローン | メインエージェント（Antigravity）と同等の権限を持ち、タスクを別コンテキストで並行処理します。 |

### 4. お問い合わせセクションの非表示化（コメントアウト）

* **依頼内容**: SEO対応。
* **対応内容**:

  * ソーシャルメディアで美しく共有されるための **OGPタグ (Open Graph Protocol)** および Twitter カードメタデータを追加。
  * **見出し構造（Heading Hierarchy）の適正化**：技術スタックの見出しを `h4` から `h3` へ昇格させ、フッターのナビゲーションタイトルを `h4` から `<div class="footer-title">` に変更（それに伴い [style.css](file:///home/devstaff/workspace/co/src/style.css) も調整）。
  * ロゴ SVG に `role="img"` と `aria-label="AstraNexus ロゴ"` を追加し、アクセシビリティ（スクリーンリーダー対応）を向上。

## 利用可能なスラッシュコマンド
開発をさらに効率化するために、チャットUI上で以下のスラッシュコマンドをエージェントに推奨・実行させることができます。

* `/goal`: 夜間などの長時間タスクを実行する際、エージェントに徹底的なタスク遂行を命じるコマンド。
* `/schedule`: 特定のタスクを定期実行したり、ワンタイムタイマーを設定したりするコマンド。
* `/grill-me`: 設計方針や要件について、エージェントと対話型インタビュー形式で合意を形成するコマンド。
